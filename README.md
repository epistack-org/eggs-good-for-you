# eggs-good-for-you

**Are eggs good for you — the egg/CVD meta-analyses**

Self-contained, verify-only mirror of the `eggs-good-for-you` case bundle (CASE-REPO-SPEC.md). It ships 15 pre-minted `records/` in the pinned assembly order (`CASE.json` `records`), a `PROVENANCE.md` vetting record, an `engine.pin` and 5 version-of-record source excerpt(s) under `sources/`, and an `expected.digest`. Verify standalone with `cairn cases verify .` and recompute the digest against `expected.digest`; no `build.py`, no signing seed, no minting required.
